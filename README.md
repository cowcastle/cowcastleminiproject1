# cowcastleminiproject1

+ ### mini project 주제
  출퇴근을 관리하는 시스템

+ ### mini project 에 대한 소개
  mini project - "**SRC checker**"는 출근시간과 퇴근시간을 관리하기 위해 구현된 시스템이다.(SRC는 super rapid commute의 축약임)
